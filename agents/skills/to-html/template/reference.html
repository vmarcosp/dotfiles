Section rhythm

Wrap top-level sections in .th-section. Wrap each H3 subsection in .th-subsection. Use .th-stack to space mixed children. Never set Tailwind margin utilities on section wrappers — spacing is centralised here so artifacts have a consistent vertical rhythm.

Term list (glossary)

Foundation draft
The in-progress brand-identity content during a run. Stored as a CP singleton on the run's branch; promoted by merging into main.
Active run
The run currently surfaced by the front-end. MVP exposes exactly one; back-end allows N concurrent.
Spec draft
The multi-page generation plan produced by the agent before any schema is emitted, scoped per run.

example.ts
type ValidationResult =
  | { ok: true;  value: ValidatedPageSchema }
  | { ok: false; errors: ValidationError[] };

function validatePageSchema(
  schema: unknown,
  catalog: Record<string, CatalogEntry>,
): ValidationResult {
  // ...
}
source 
<div class="th-code-wrap" x-data="codeBlock">
  <div class="th-code-label">example.ts</div>
  <pre class="th-code"><code class="language-ts">
    type ValidationResult = …
  </code></pre>

configBlock

A live form scope. Each key in the initial object becomes a reactive property — bind inputs with x-model and reference them in x-text spans or in a copy() template string with {{key}} placeholders.
postgresql://@localhost:/
source 
<div x-data="configBlock({port: '5432', user: 'postgres', db: 'app'})">
  <input x-model="port" />
  <input x-model="user" />
  <input x-model="db" />

  <pre>postgresql://<span x-text="user"></span>@localhost:<span x-text="port"></span>/<span x-text="db"></span></pre>

  <button @click="copy('postgresql://{{user}}@localhost:{{port}}/{{db}}')"
          x-text="copied ? 'Copied' : 'Copy connection string'"></button>
</div>

Auto-collapses when its text content exceeds 200 characters. Short prompts stay expanded; long ones get a toggle. Use in the page header to show the original prompt that generated the artifact.

tabs

Single-active-tab state. Pass the initial active tab as an argument, or omit it to default to the first [data-th-tab] child. Tab triggers call set('name'); panels use x-show="is('name')".

A short summary of what this thing does and why it exists.

const result = compute(input);
return result;
  • Edge case: empty input returns null.
  • Performance: O(n log n) on the input length.
